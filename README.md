# Topology Doctor

[[Topo Doctor.py](https://github.com/user-attachments/files/24929375/Topo.Doctor.py)](https://github.com/user-attachments/assets/1287704b-290c-47cf-a964-a751143b2469)

Download Extension: (https://extensions.blender.org/add-ons/topology-doctor/)
Topology Doctor is a professional analysis tool for Blender that helps you create clean, production-ready 3D models.

## Features
* **One-Click Analysis:** Instantly checks for N-Gons, Holes (Non-Manifold), and Slivers.
* **Health Score:** Gives your model a rating out of 100.
* **Interactive Focus:** Click the magnifying glass ðŸ” next to any error to zoom directly to the bad geometry in Edit Mode.
* **Cycle Mode:** Click repeatedly to jump through every error one by one.

## Installation
1.  Download the `.zip` file.
2.  Open Blender.
3.  Go to **Edit > Preferences > Get Extensions**.
4.  Click the arrow in the top right and choose **"Install from Disk..."**.
5.  Select the zip file.

## Usage
1.  Press **N** in the 3D Viewport to open the sidebar.
2.  Click the **Topo Doctor** tab.
3.  Select a mesh and click **Analyze Mesh**.
